# MERN Full Stack Project

This is a full stack web application built using the MERN stack.

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js

## Project Structure
client/ - React frontend  
server/ - Node.js and Express backend  

## Installation

1. Clone the repository

git clone https://github.com/yourusername/mern-fullstack-project.git

2. Install server dependencies

cd server  
npm install

3. Install client dependencies

cd ../client  
npm install

4. Run the project

Backend:
cd server  
npm start  

Frontend:
cd client  
npm start  

## Features
- User authentication (JWT)
- REST API
- MongoDB database
- React frontend
